
# Cardápio Digital - Botekão Lanches

![Banner do Botekão Lanches](assets/img/banner.jpg)

Sistema de cardápio digital interativo para o Botekão Lanches com painel de gerenciamento, controle de mesas e delivery.

## 🚀 Funcionalidades

- Cardápio digital com fotos em alta qualidade
- Divisão por categorias (hambúrgueres, bebidas, etc.)
- Sistema de pedidos para 10 mesas
- Área de delivery com formulário completo
- Painel de gerenciamento para administradores
- Integração com WhatsApp e impressora térmica
- Checkout com múltiplas formas de pagamento
- Totalmente responsivo para mobile

## 🛠️ Tecnologias

- HTML5, CSS3 e JavaScript
- [Font Awesome](https://fontawesome.com/) - Ícones
- [Google Fonts](https://fonts.google.com/) - Fontes tipográficas
- [Chart.js](https://www.chartjs.org/) - Gráficos no painel
- Netlify - Hospedagem

## 📦 Estrutura do Projeto

```
botekao-cardapio/
├── index.html          # Página principal

├── checkout.html      # Página de pagamento

├── assets/
│   ├── css/          # Estilos
│   ├── js/           # Scripts JavaScript
│   └── img/          # Imagens do cardápio
└── README.md          # Este arquivo
```

## 🔧 Instalação

1. Clone o repositório:
   ```bash
   git clone https://github.com/wak1lon/botekaoomelhorcardapio.git

   ```

2. Para desenvolvimento, basta abrir o `index.html` em qualquer navegador

3. Para produção, faça deploy no Netlify:
   - Conecte sua conta do GitHub
   - Selecione este repositório
   - Clique em "Deploy site"

## 🌐 Acesso Online

O sistema está disponível em:

## ⚙️ Configuração

Edite os seguintes arquivos para personalizar:

- `assets/js/main.js`: Configure IDs do GA4 e Facebook Pixel
- `gerencia.html`: Ajuste as credenciais de login
- `assets/img/`: Adicione suas imagens (1080x1350px recomendado)
